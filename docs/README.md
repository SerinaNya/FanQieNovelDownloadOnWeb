# FanQieNovelDownloadOnWeb
一个用以下载番茄小说的web应用  
您可在[这里](https://github.com/weiwei-cool/FanQieNovelDownloadOnWebUI)找到此项目的前端

## 特点

 - 优秀的django python网络服务应用
 - 快速的docker部署
 - 支持amd和arm两种架构
 - 支持txt、epub格式保存
